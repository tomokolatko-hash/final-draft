​This assignment was so far the most interesting project for me because if the code I wrote was correct, the website would turn out just the way I imagined. :) I liked how I can test it right away too! The code looked a little bit complicated and intimidating in TextEdit, but when I used W3 Schools, I was able to write the code properly and it was fun! However, I realized that it's important to be careful, because things like using capital letters in file names or making small mistakes in the file name can cause the site not to display correctly. I studied that I will have to be more careful for spelling and typos when I code. For this new HTML code, I added a font change, font size, corrected image sizes, and add grey line to the top text and add some links. I had never done any programming before, so this was a great opportunity for me. I’d like to continue studying HTML, take good photos of my ceramics, and eventually create a website to showcase and hopefully sell my work.
